    public withExtension(key: uri | string, value: *): ActivityBuilder source

    Add an extension to the Activity definition

    If the key is not a URI but matches a name of an Activity Extension in a Profile loaded by StatementBuilder#withProfile, the uri of that Activity Extension will be used. Throws an error if the key does not match a name and is not uri-like.

    NameTypeAttributeDescription
    key uri | string

    URI key of the extension
