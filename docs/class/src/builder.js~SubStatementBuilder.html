    SubStatementBuilder

    Extends:

    _BuilderRecord → BuilderRecord → StatementBuilder → SubStatementBuilder

    This class is returned when StatementBuilder#asSubStatement is called. It makes sure the disallowed SubStatement properties (id, timestamp, version, and authority) are not set.
